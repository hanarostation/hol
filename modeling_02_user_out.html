Mission 02

소환사님, 하나로 협곡에서 데이터 과학의 핵심을 다루는 새로운 퀘스트가 도착했습니다.

이번 미션은 가구 매장에서 관리하는 회원 데이터를 이용해 회원 탈퇴 여부를 예측하고,
협곡의 고객 유지 전략을 설계하는 중요한 여정입니다.

가구 매장은 단순히 제품을 판매하는 공간을 넘어,
회원들의 생활과 밀접하게 연결된 서비스를 제공해야 지속적인 성장을 이룰 수 있습니다.

이번 퀘스트는 데이터를 통해 고객 이탈 가능성을 사전에 예측하고,
이를 바탕으로 고객 관리 전략을 세우는 데 초점이 맞춰져 있습니다.

📌 제출 방식

· 학습 데이터로 모델을 먼저 생성한 뒤 파일로 저장
· 파일을 가져와 검증 데이터를 모델에 집어넣는 것까지 코드 작성
· 학습된 모델 파일과 검증 데이터 평가 코드 파일 두 가지를 제출
