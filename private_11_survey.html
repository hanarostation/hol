컨설팅의 탑 — 데이터를 해독하라!
Private Mission

하나로 협곡의 중심에는 협곡 내 모든 기업의 성장과 발전을 돕는 컨설팅의 탑이 있습니다. 이 탑에서는 기업 경영, 조직 개발, 전략 수립 등을 지원하기 위해 컨설팅사가 수행한 다양한 프로젝트에 대한 피드백이 모이고 분석됩니다.

⚡ 퀘스트 브리핑
최근 컨설팅사의 성과를 평가하는 설문조사가 진행되었고,
이 데이터를 바탕으로 협곡의 기업 성장 전략을 새롭게 구상하기 위한
분석 퀘스트가 소환사님께 주어졌습니다!

협곡 내 266명의 그룹원이 설문에 참여했으며, 각 질문에 대해 1점에서 5점 사이의 만족도를 남겼습니다.

📊 이 데이터는 단순한 점수의 집합이 아닙니다

· 협곡 내 기업들이 무엇을 필요로 하는지
· 컨설팅사가 어떻게 기여했는지
· 앞으로 나아갈 방향은 무엇인지에 대한 단서를 담고 있습니다.
📋
설문 분석 — 7가지 임무를 완수하라!
Mission Tasks
📂 데이터 파일 — Survey_data.xlsx
📌 데이터 구조 안내

· '변수'/'요인'/'측정변수'/'번호'/'CODE' → 설문 항목 Column
· 숫자로 구성된 Column (1~266) → 설문에 참여한 그룹원 ID
· 아래 숫자 → 1~5점 설문 만족도 점수
· 일반사항의 성별: 남성-1 / 여성-2, 연령: 2→20대 / 3→30대 / 4→40대 ...
  1. 1
    위의 데이터를 가져와, 데이터를 아래와 같이 변환하시오.
    • '변수' → '항목' / '요인' → '세부항목' 으로 항목명 변환
    • 그룹원 ID는 새로운 'ID'라는 변수로 정리 / 그룹원이 입력한 설문 값은 '점수' 라는 변수로 정리
    • 맨 아래 Row는 각 설문에 참여한 그룹원의 분류 항목이다.
    • '항목'의 일반사항은 해당 설문자의 고유 특징을 나타낸 항목으로, 별도로 Column을 뽑아 정리
    • 변환 뒤 데이터는 Row 12,236 / Columns 18개
  2. 2
    '항목' 별 '점수'의 대표값 차이가 있는지 검정하고자 한다. 각 '항목' 별 '점수'의 평균을 계산하고, '컨설턴트의 역량'과 '경영컨설팅 성과'의 점수 대표값의 차이가 있는지 검정하시오.
  3. 3
    '항목'에서 '컨설팅사의 특성' 중, '명성'과 '전문성'항목의 '점수'가 서로 상관성이 있는지 가설검정을 수행하시오.
  4. 4
    '항목'에서 '경영컨설팅 성과' 중, '실행성'과 '효과성'항목의 '점수'가 서로 상관성이 있는지 가설검정을 수행하시오.
  5. 5
    '성별'에 따라 '컨설턴트 역량'의 '공통역량'항목 점수의 대표값 차이가 있는지 검정하시오.
  6. 6
    '연령대' 별 '경영컨설팅성과' 점수의 평균 분포가 어떻게 달라지는 지 확인하고자 한다. 각 연령대의 '경영컨설팅성과' 점수 평균을 히스토그램으로 표현하시오.
  7. 7
    설문자 정보에 따라, 설문조사 점수의 총점을 예측하고자 한다. 위의 데이터를 ID 기준으로 정리하여, ID 기준 점수의 평균을 계산한 뒤, 해당 검사자의 '매출규모' / '성별' / '업종' / '직위' / '종업원수' / '직장경력' / '컨설팅분야' / '컨설팅 비용' / '학력' / '연령대'을 입력했을 때, 해당 입력값에 대한 설문 점수의 평균을 예측하는 모델을 구성하시오.
    • 학습 검증 데이터 비율 8:2
    • R² 와 MSE로 학습 성능 / 일반화 성능 평가 실시
