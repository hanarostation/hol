협곡 유통망의 비효율을 바로잡아라!
Private Mission

하나로 협곡에는 의료 제품을 도매로 공급받아 협곡 곳곳의 소매점과 병원에 납품하는 유통업체들이 있습니다. 이들은 협곡 주민들의 건강을 지키는 중요한 역할을 맡고 있습니다.

하지만 최근 들어 협곡의 유통망에 불균형과 비효율이 발생하면서, 협곡 주민들의 신뢰를 회복하기 위한 분석이 절실해졌습니다.

⚡ 퀘스트 브리핑
도매 가격, 판매 금액, 업체 분류, 납품 수량 등
모든 정보가 협곡 내 유통 흐름의 한 조각을 이루고 있습니다.
이 데이터가 제대로 분석되지 않으면 주민들의
의료 제품 접근성에 악영향을 미칠 것입니다!

이를 해결하기 위해 협곡 데이터 분석팀이 출동했습니다.

🎯 이번 퀘스트의 목표

데이터를 활용해 협곡 유통의 비효율을 바로잡고,
효율적이고 공정한 유통망을 설계하는 것입니다.
📋
분석 요청 — 8가지 임무를 완수하라!
Mission Tasks
📂 데이터 파일 — WS_Medical_data.csv
  1. 1
    위의 의료용품 데이터를 가져와, 업체분류의 빈도수가 높은 10개 데이터들만 추출하여 뽑아 업체분류에 따른 23년 대비 24년 판매금액 증감율을 계산하시오.
  2. 2
    도매최대가와 도매최소가의 차이가 가장 큰 제품을 확인하고, 해당 제품이 어떤 업종에 가장 많이 납품되었는지, 업체분류 별 도매매입수량의 합을 계산하시오.
  3. 3
    업체유형에서 "개인병원"에 해당하는 데이터를 뽑아, 시간에 따른 도매매입수량의 합이 어떻게 변화하는지 선그래프로 시각화 하시오.
  4. 4
    각 연도 별 판매금액이 가장 많은 팀을 확인하고, 해당 팀의 24년 대비 25년 판매금액 평균(또는 대표값)의 유의미한 차이가 있는지 검정하시오.
  5. 5
    판매금액(납품가격)보다 도매매입금액이 더 큰 (마진이 남지 않는) 데이터를 뽑아 제품의 빈도수 비율을 확인하고, 전체 데이터에서 "마진여부" 항목을 만들어 마진이 있는 데이터는 1 / 없는 경우는 0으로 분류하시오.
  6. 6
    '마진여부'를 분류하는 분류모델을 구성하고자 한다. 아래의 데이터를 활용해, 분류모델을 만들고 학습데이터와 검증데이터의 성능을 확인하시오.
    • 입력 데이터 : '팀' / '업체분류' / '도매최대가' / '도매최소가' / '영업그룹' / '연도' / '월'
    • 학습 / 검증 데이터 비율은 7:3
    • Threshold Tuning 실시 및 AUC 값을 활용하여 평가 실시
  7. 7
    '거래예측가격'(팀에서 정한 예측가격, 또는 거래 목표)과 '판매금액'(실제 판매된 가격)을 비교하여, 예측가격을 달성하지 못한 데이터를 뽑아, 해당 데이터의 '팀' / '업체분류' / '제품명'의 빈도수를 확인하시오.
  8. 8
    거래예측가격(또는 거래 목표 금액)을 정확하게 추정하기 위한, 판매금액 예측 회귀모델을 만들고 학습데이터와 검증데이터의 성능을 확인하시오.
    • 입력 데이터 : '업체분류' / '도매최대가' / '도매최소가' / '연도' / '월' / '제품명'
    • 학습 / 검증데이터 비율은 7:3
    • MSE와 R²를 이용하여 평가 실시
