소환사님, 하나로 협곡에 중요한 데이터 분석 미션이 도착했습니다.

이번 퀘스트는 A어학원의 데이터를 분석하여 강사별 성과를 파악하고 수업 운영의 효율성을 극대화하는 데 목적이 있습니다.

⚡ 퀘스트 브리핑
단순히 데이터를 다루는 작업이 아닙니다.
강사들의 노력을 정량적으로 분석하고
어학원의 운영 전략을 설계하는 도전입니다!

첫 번째 과제는 다양한 언어 수업 중 가장 높은 총지급 강사료를 기록한 수업을 확인하는 것입니다. 이는 어학원에서 가장 가치 있는 언어 수업을 파악하고, 해당 수업에 더 많은 자원을 투입할 전략을 설계하는 중요한 단서가 될 것입니다.

강사별 분석도 핵심 과제 중 하나입니다. 전체 데이터를 통해 가장 많은 강사료를 받은 강사를 확인하고, 해당 강사의 총 강의 기간을 계산해야 합니다. 이를 통해 강사의 성과와 어학원에 미친 영향을 구체적으로 파악할 수 있습니다.

📋
분석 요청 — 9가지 임무를 완수하라!
Mission Tasks
📂 데이터 파일 — language_classes.csv
  1. 1
    위의 데이터를 불러와, 수업이 진행되는 언어 중 ('언어(수업)') 가장 "총지급 강사료"의 합이 높은 수업을 확인하시오.
  2. 2
    각 "담당 매니저" 별 "클래스 수업료"와 "총지급 강사료"의 평균을 계산하시오.
  3. 3
    전체적으로 볼 때, 가장 많은 강사료를 받은 강사를 확인하고, 해당 강사의 총 강의 기간을 계산하시오.
  4. 4
    각 강사들의 강의 빈도를 계산하고, 강사 별 "총 강사료"의 합과 가장 최근에 강의한 일자를 확인하여, 각 강사의 RFM Score를 계산하시오.
    • 각 강사 별로 RFM Score 산출
    • "강의 빈도" - Frequency / "총 지급 강사료"의 합 - Momentary / "가장 최근에 열린 강의" (마지막 일로부터, 해당 클래스 강의까지 일수) - Recency
    • RFM Score = 0.3 x Frequency + 0.5 x Momentary - 0.2 x Recency
  5. 5
    앞서 계산한 RFM 분석으로 강사들의 등급을 분류하시오.
    • RFM Score 상위 10% → A 등급
    • RFM Score 상위 30% → B 등급
    • RFM Score 상위 50% → C 등급
    • RFM Score 상위 50% 미만 → D 등급
  6. 6
    각 등급 별 "총 지급 강사료"의 평균과 "강의 빈도"를 시각화하시오.
  7. 7
    "class 종류"에서 빈도수가 높은 상위 4개의 항목에 대해 "언어(수업)"에서 영어, 일본어, 중국어의 유의미한 차이가 있는지 가설 검정을 수행하시오.
  8. 8
    강의 기간이 가장 긴 클래스 상위 100가지를 확인하여, 해당 강의들의 "언어(수업)" 비율을 계산하시오.
  9. 9
    강의 정보를 입력했을 때, 클래스 수업료를 예측하는 회귀 모델을 구성하시오.
    • 강의 기간 / 강사명 / 담당매니저 / 언어(수업) / 강사료 / class 종류를 입력 받아야 함
    • 특성 공학 기법 활용
    • 학습/검증 데이터 비율은 7:3
    • 검증은 R²를 활용하여 Test Set 성능 확인
